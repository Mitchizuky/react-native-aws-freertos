# react-native-aws-freertos

React native aws freertos bridge

## Installation

```sh
npm install react-native-aws-freertos
```

## Usage

```js
import AwsFreertos from "react-native-aws-freertos";

// ...

const result = await AwsFreertos.multiply(3, 7);
```
